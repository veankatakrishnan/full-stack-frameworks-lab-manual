# Experiment 04 - Simple Calculator

A fully functional calculator application built with ReactJS that performs standard arithmetic operations.

## Features

- **Basic Arithmetic Operations**: Addition, Subtraction, Multiplication, Division
- **Additional Functions**:
  - Clear (C) - Reset calculator
  - Backspace (⌫) - Delete last digit
  - Percentage (%) - Convert to percentage
  - Toggle Sign (±) - Change positive/negative
  - Decimal Point (.) - Support decimal numbers

## Operations Supported

- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)

## How to Use

1. Click number buttons to enter values
2. Click operation buttons (+, −, ×, ÷) to perform calculations
3. Click equals (=) to see the result
4. Use C to clear and start over
5. Use ⌫ to delete the last digit

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

## Technologies Used

- React 19.2.0
- Vite 7.2.4
